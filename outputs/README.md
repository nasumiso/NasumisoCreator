# outputs/ - 生成結果管理

## 概要

Windows環境のAUTOMATIC1111 WebUIで生成した画像とログを保存

## フォルダ構成

```
outputs/
├── test_generations/      # テスト生成画像
│   └── nasumiso_v1/      # プロジェクトごとに整理
├── prompts/               # プロンプトテンプレート
└── generation_logs/       # 生成パラメータログ
```

## 使用方法

Windows環境で生成した画像を、プロジェクトごとに整理して保存してください。
