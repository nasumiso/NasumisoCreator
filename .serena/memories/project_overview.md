- Purpose: NasumisoCreator is a Python-based assistant for recreating illustrator なすみその art style via LoRA training. It provides a Gradio WebUI plus CLI scripts to resize images, auto caption with WD14, add common tags, and manage training assets aimed at Stable Diffusion/kohya_ss + Google Colab workflows.
- Tech stack: Python 3.10+, Gradio 4.x, Pillow, WD14 Tagger via onnxruntime, numpy/pandas, huggingface-hub (no backend server aside from the Gradio app). Runs on macOS/Linux; future plans include Google Drive API integration.
- Architecture & structure: `app.py` is the main entry (Gradio UI with logging in `logs/`). `scripts/` holds preprocessing utilities (resize/rename, auto caption, add_common_tag plus helpers). `projects/` stores dataset stages (raw, processed, tagged, models). `notebooks/` contains Colab notebooks (for training). `config/` holds JSON config templates; `.claude/spec` documents requirements and specs. Shell scripts (`start_nasumiso_trainer.sh`, `stop_nasumiso_trainer.sh`) wrap the UI. No backend database; state kept in filesystem dirs under `projects/`.
- Key workflows: Tab 1 handles preprocessing (resize/tag); Tab 2 edits tags; Tab 3 (planned) handles training/Drive; Tab 4 stores settings like SD WebUI paths and training mode.
- Data/config: Copy `config.json.example` to `config.json` to set SD WebUI paths, default project dirs, image size, caption threshold, and tags.
